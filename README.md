## Dasbord-1 -Sales & Profit Analytics Dashboard

Project Overview
This Power BI dashboard provides a comprehensive analysis of global sales and profitability. It is designed to help stakeholders identify which product categories and regions are driving growth and where the business can optimize for higher margins.

Key Features
KPI Tracking: Real-time visibility into total Sum of Profit (28.36K) and Sum of Sales (126.68K).

Category Analysis: Deep dive into Furniture, Office Supplies, and Technology performance.

Regional Insights: Breakdown of profit trends across different global regions and countries (e.g., Afghanistan, Australia, Brazil).

Interactive Filters: Users can slice data by Category, Sub-Category, and Country to find specific data points instantly.

Technical Tools Used
Power BI Desktop: For data modeling and visualization.

DAX (Data Analysis Expressions): Used for calculating "Max of Profit" and "Max of Quantity" metrics.

Data Transformation: Cleaned and structured raw data to ensure accurate reporting.

Visualizations Explained
Donut Chart: Shows the percentage distribution of profit by category (Office Supplies leading at ~33.75%).

Funnel Chart: Visualizes the profit contribution across different product tiers.

Waterfall Chart: Tracks the "Sum of Profit by Region," highlighting which areas see increases or decreases in earnings.

Data Table: Provides granular details on Sub-Categories like Bookcases, Chairs, and Appliances.

## Dashbord 2 -Student Performance & Placement Analytics Dashboard

Project Overview
This interactive Power BI dashboard provides a comprehensive analysis of student academic records, internship experiences, and placement outcomes. The goal is to identify patterns between academic performance (CGPA, IQ, Communication Skills) and successful job placements across various college IDs.

Key Features & Visualizations
Placement Overview: A donut chart showing the distribution of placed vs. non-placed students (83.41% placement rate).

Academic Benchmarking: A heat-map table tracking Maximum CGPA and Academic Performance by College ID.

Skill Impact Analysis: A scatter plot analyzing the relationship between IQ, Academic Performance, and Communication Skills.

Experience Tracking: A stacked column chart visualizing the count of Internship Experiences and Projects Completed per college.

Communication Breakdown: A horizontal stacked bar chart comparing communication scores across different CGPA brackets.

Technical Stack
Tool: Power BI Desktop

Data Source: CSV/Excel (Student Dataset)

DAX: Used for calculated measures like Sum of CGPA and Count of Placements.

Data Cleaning: Power Query for handling null values and formatting College IDs.

Insights Derived
Placement Correlation: High placement rates are strongly linked to the number of projects completed and internship experience.

Performance Consistency: Most colleges maintain a maximum CGPA above 10.0, showing high academic standards.

Communication Skills: There is a diverse range of communication skills across colleges, which significantly impacts the "Academic Performance" metric.
